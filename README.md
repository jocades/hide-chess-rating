# Hide Chess Rating

Chrome extension to hide online chess ratings in popular sites, win your next game with a clearer mind.

---

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.1.17. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
